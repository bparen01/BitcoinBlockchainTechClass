# BitcoinBlockchainTechClass
This repository contains slides for an introductory Computer Science class on Bitcoin, Blockchain, and Cryptocurrency technologies.
The slides have been used for a graduate class, though could be used for an undergrad class as well.
The material is meant be an overview of blockchain and cryptocurrency technologies.
The first part focuses primarily on Bitcoin as the original cryptocurrency, then continues with Ethereum and other topics.
When used in a class, there would be additional material including demos and assignments.
